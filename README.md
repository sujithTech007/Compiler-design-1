# Compiler Assessment

This repository contains a simple Python script for a compiler assessment. 

## Features
The `compiler_assessment.py` file includes:
1. **Lexer**: Tokenizes a simple arithmetic expression involving variables and numbers.
2. **Parser**: Generates a basic Abstract Syntax Tree (AST) representing binary operations.
3. **Semantic Analyzer**: Checks if the left-hand side of an assignment is a valid variable name.

## Usage
Run the script using Python:

```bash
python compiler_assessment.py
